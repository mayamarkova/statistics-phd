Модул 1 · Подготовка на данните
Казус: "Намери грешката в SPSS"
⏱️ 15 мин. 🕵️ Детективски 👥 Групи от 2 души
📋 Сценарий
Колегата ви Елена ви показва Variable View в SPSS и казва: "Не разбирам защо резултатите ми от t-теста изглеждат странно." Намерете 5-те грешки в настройките на нейните променливи.
🔍 Variable View — как изглежда в SPSS
Name Type Decimals Values Missing Measure
ime String 0 None None Nominal
vazrast String 0 None None Nominal
pol Numeric 2 None None Nominal
obrazov Numeric 0 1, 2, 3 None Scale
uspeh Numeric 2 None None Scale
🎯 Задача

Намерете 5-те грешки в таблицата. Първата група с всички правилни отговори печели!

Покажи решение
5-те грешки:
  1. vazrast е String — трябва да е Numeric (иначе не можем да правим математически операции с нея)
  2. vazrast е Nominal — трябва да е Scale (възрастта е числова)
  3. pol има 2 десетични — безсмислено за категория (1.00 за мъж?); трябва да е 0
  4. pol няма Value Labels — трябва да се дефинира 1 = Мъж, 2 = Жена
  5. obrazov е Scale — трябва да е Ordinal (основно/средно/висше имат ред)
  6. uspeh няма дефинирани Missing values — обикновено се кодира 999 или 99 като Missing
Защо е важно: Ако Measure е зададен грешно, SPSS ще позволи да правите "средна стойност на пола" или "стандартно отклонение на образованието" — глупости, които могат да попаднат в дисертацията.
